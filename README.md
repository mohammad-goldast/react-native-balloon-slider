# react-native-balloon-slider

It is a ReactNative slider for the #UXChallenge1 challenge and you can use it now.

    <BalloonSlider
	  onChange={(value)  =>  console.log('onChange: ',  value)}
      onChangeComplete={(value)  =>  console.log('Complete: ',  value)}
	/>
